## Списки

Чтобы добавить ненумеровнные списки, необходимо пункты выделить (*). Например, вот так 
* элемент 1
* элемент 2

Чтобы добавить нумерованные списки, необходимо пункты просто нумеровать.
Напрмер, вот так:
1) элемент
2) элемент
